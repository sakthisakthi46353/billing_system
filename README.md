# Billing System (Django Project)

## Project Overview
This is a Billing System web application developed using Django.
It helps manage customers, products, invoices, payments, and reports
in a simple and organized way.

---

## Features

### Customer Management
- Add new customers
- Edit and delete customers
- View customer statements

### Product Management
- Add products
- Edit products
- Delete products
- Manage product prices and stock

### Invoice Management
- Create invoices with multiple products
- Automatic subtotal, tax, and total calculation
- Invoice status tracking (UNPAID, PARTIALLY_PAID, PAID)
- Automatic stock reduction
- View detailed invoice information

### Payment Management
- Add payments to invoices
- Support partial and full payments
- Automatic balance calculation
- Payment history tracking

### Reports
- Customer Balance Summary
- Sales Summary
- Top Selling Products
- Customer Statement

---

## Technology Stack
- Python 3
- Django Framework
- SQLite Database
- HTML & CSS
